# Cheat-sheet-and-Guide-for-Game-Developers

This repository serves as a guide and cheatsheets for Game Developers whether new, or unfamiliar with shortcuts, how to install something, and other essentials needed in the game development loop. 

This will only go over these tools specifically for Game Development, so if you are using it for anything other than that please refer to [Useful Links](#useful-links) for useful websites.

**New to this repo? Start at the [Table of Contents](#table-of-contents) to find what you directly need.**

Currently a huge work in progress.

## What does this repository go over?
Each topic will include installation guides, useful usage tips for your projects within these engines and software, and helpful shortcuts which can improve your Quality of Life.

It may be expanded in the future by myself or others to other topics which have not been covered in the base, original ones.
